Inline 3D in the page

CEF offscreen-render → the host drives the real display-processor weave through XR_EXT_weave. The host never weaves; the 3D element below is a side-by-side canvas whose flat pixels are replaced by the woven 3D in the host's present surface. Scroll, zoom, and drag the window — the lattice must stay locked.

Above the element

This is ordinary 2D DOM content forming the surround. It scrolls normally. The 3D element participates in page flow, so scrolling moves it — the host tracks its committed device-pixel rect every frame and feeds it to the weave RPC as the window-relative rect.

As the viewer's head moves, the runtime's tracked eyes flow back to this page, which re-renders the element off-axis (look-around) using an asymmetric-frustum projection.

Below the element

More surround text so the page is comfortably scrollable. Use the mouse wheel or Page Down to scroll the 3D element across the viewport while watching the live display for any lattice drift or 3D collapse.
